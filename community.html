Hyperdash: Vanguard Clan Discord

Hyperdash is an up-and-coming Oculus Quest VR game much like Overwatch or Quake, with the ability to slide on rails, teleport-dash and dual-weild a variety of weapons. So far the game is in alpha and yet feels genuinely more fun and polished than most existing Quest games.

You'll need to install the game using Sidequest, its currently in Alpha 5 and had added a host of features like a new running mechanic and QOL improvements.

Join Sloga Group's clan Vanguard (VGRD) and participate in community held tournements and scrims along with private clan matches. We are looking for any and everybody to build our numbers so we can form a full team.
